//Create a nuget file
1.	Right click on the project.
2.	Select the option "properties".
3.	Select the tab "package".
4.	Fill the form.
5.	Rebuild the solution.
6.	Rigth click on the project.
7.	Select the option "pack".

//Publish a nuget file to nuget.org
1.	Sign up in https://www.nuget.org/
2.	Select the top tab "Upload".
3.	Browse your file and upload.
